# Typst Template DHBW

This template is based on the [typst-dhbw-template](https://github.com/DannySeidel/typst-dhbw-template/tree/main) but customized with a different appearance, author's declaration, confidentiality statement, and other settings. It could be used for both german and english.

## Usage
1. (VSCode) install the [Tinymist Typst](https://marketplace.visualstudio.com/items?itemName=myriad-dreamin.tinymist) Extension.


## Project Structure

### Main File
- **main.typ**: This file sets all variables such as author, language, etc. You can include multiple authors if the work is conducted by more than one person. 
- All chapters are included here.
- [change cite style](https://typst.app/docs/reference/model/cite/) 

### Chapters Directory
- Ensure each chapter is stored in its own file within this directory.

### Assets Directory
- Store all images and other assets in this directory.

### Template Directory
- Contains all settings and information necessary for the appearance. If you don't need to change anything specific, then don't touch it! :)
